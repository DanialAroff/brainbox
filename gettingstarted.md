1. Install Python 3.9 or higher from [python.org](https://www.python.org/downloads/).
2. Install LM Studio by following the instructions at [lmstudio.ai](https://lmstudio.ai/).
3. Start LM Studio and load `text-embedding-nomic-embed-text-v1.5`. If it is not available, you can download it from the LM Studio model repository.
4. Install ChromaDB by following the instructions at [chroma.com/docs/quickstart](https://chroma.com/docs/quickstart).
5. Start the ChromaDB server with the command:
   ```bash
   chroma run --path "M:\EBANK-DOC\Mobile Platform 2014\Plan\RCE\InfoHub"
   ```
